# server-code
使用python后端框架Django搭建服务器

## 安装方法
以下操作在ubuntu16.04系统的服务器下进行

- 安装python3.5以上的版本
- 用`pip3 install django`安装django2.2
- 安装mysql-8.0.15
- 用`pip3 install PyMySQL`安装PyMySQL操作数据库
- 安装mysqlclient操作数据库
